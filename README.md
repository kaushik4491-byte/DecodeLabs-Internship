## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/kaushik4491-byte/DecodeLabs-Internship.git
```

2. Navigate to the project folder:

```
cd DecodeLabs-Internship
```

3. Run the chatbot:

```
python chatbot.py
```
